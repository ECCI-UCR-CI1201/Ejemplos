
# Ejemplos del curso Programacion II

Este repositorio contiene todos los ejemplos vistos en el curso Programacion II durante el I semestre del 2015. 

A medida que el semestre avanza se irán agregando más ejemplos.

Para cambiar el proyecto activo que se ejecuta en Visual Studio 2013: 
1. Hacer click derecho en el proyecto respectivo
2. Hacer click en *Set as StartUp Project*
